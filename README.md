# Atividade React - Componentes

Projeto desenvolvido para a disciplina utilizando React.

## Objetivo

Praticar a criação e utilização de componentes em React.

## Funcionalidades

- Componente reutilizável
- Estilização moderna com CSS
- Card de perfil interativo
- Contador de curtidas
- Tecnologias exibidas dinamicamente

## Tecnologias utilizadas

- React
- JavaScript
- CSS

## Como executar o projeto

Clone o repositório:

```bash
git clone LINK_DO_REPOSITORIO
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```
